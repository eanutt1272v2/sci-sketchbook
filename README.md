# p5.js and Processing Sketch Library

This repository contains a collection of primarily computational p5.js and Processing sketch files and resources, organised into individual project folders.

## Up and Running

### p5.js Sketches
To run any of the p5.js sketches, navigate to the respective project folder and open the `index.html` file in a web browser. Ensure that all related `.js`, `.css`, shader (`.glsl`), and `.json` (if applicable) files are in the same directory or correctly linked together within the `index.html` and `sketch.js` files.

### Processing Sketches
To run any of the Processing sketches, you will need to have the Processing Development Environment (PDE) installed on your computer. You can download the latest version from the official Processing website [1].

Once Processing is installed:
1. Navigate to the desired sketch folder within this repository.
2. Open the `.pde` file (the main sketch file) using the Processing IDE.
3. Click the "Run" button (a triangular play icon) in the Processing IDE to execute the sketch.

Make sure that any associated data files, libraries, or assets are correctly placed within the sketch folder or linked as required by the Processing sketch.
