# Mandelbulb GLSL Shader

## Concept
High-performance 3D Mandelbulb rendering using GPU raymarching.
Uses Signed Distance Functions (SDF) and raymarching in GLSL to render the 3D fractal.

## Controls
Mouse-driven camera rotation and zoom; real-time parameter adjustments via uniforms.

## Technology
- **Tech Stack:** p5.js & GLSL Shaders
