# Mandelbulb GLSL Shader

## Overview
Real-time Mandelbulb raymarching in GLSL, hosted in p5.js/WebGL.

## Implementation
- `sketch.js` orchestration
- `vert.glsl` and `frag.glsl` shaders

## Controls
- Mouse-driven camera interaction
- Real-time render parameter tuning via uniforms

## How to Run

### Browser (p5.js/WebGL)
```bash
cd sketchbook/Mandelbulb_GLSL_Shader
python3 -m http.server 8080
```
Open `http://localhost:8080`.
