# Neural Network

## Overview
Processing sketch demonstrating a basic feedforward digit-recognition neural network and its training behaviour in a visual, interactive format.

## Implementation
- `Neural_Network.pde`

## Controls
Training and visual parameters are controlled within the Processing sketch interface and source settings.

## How to Run

### Processing (Java)
Open `sketchbook/Neural_Network/Neural_Network.pde` in Processing 4.x and click Run.
